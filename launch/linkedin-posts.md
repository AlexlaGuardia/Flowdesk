## Post 1: Critik (post today)

Three freelancer tools displaced thousands of users in two months.

But here's the thing nobody talks about: the vibe-coded replacements people are building have security holes you could drive a truck through.

53% of AI-generated code has security vulnerabilities. 35 new CVEs in March alone from AI-assisted code.

I built Critik to catch what Copilot ships and Snyk overcharges for.

Two-pass scanning:
Pass 1: regex + AST catches hardcoded secrets, SQL injection, XSS
Pass 2: AI reviews full file context for the subtle stuff

Zero config. One command: pip install critik && critik scan

Free and open source. VS Code extension. GitHub Action. Pre-commit hook.

Try it: critik.dev
